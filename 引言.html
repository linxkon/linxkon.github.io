Agent 设计模式 - 引言

章节目录

本目录用于存放翻译后的章节文件。

文件命名规范

  • 使用格式：chapter-XX.md，其中 XX 为章节编号（01-25）
  • 例如：chapter-01.md、chapter-02.md 等

翻译流程

  1. 从 original/ 目录获取原文
  2. 在此目录创建对应的翻译文件
  3. 保持原文的结构和格式
  4. 更新图片引用路径为 ../images/agent_images/chapter-XX/

注意事项

  • 文件编码使用 UTF-8
  • 保持 Markdown 格式规范
  • 代码块使用正确的语言标识符
